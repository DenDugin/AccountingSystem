Главный класс - AccountSystem.jar

Входные данные в формате xml находятся в файле - Input.xml

Файл с перечнем свойств (названия полей для таблиц) - Properties.xml 

При изменении данных необходимо учитывать следующее :

- При добавлении нового свойства для животного в Properties.xml  необходимо также добавить данные в Input.xml с аналогичным значением тега.

- Колличество параметров должно быть одинаковым в обоих файлах.

- Набор свойств и параметров неограничен по колличеству.  


Правила (запросы) необходимые для исполнения находятся в файле - Rules.xml.

В аргументе Field необхдимо указывать название поля. Колличество правил - неограничено.
